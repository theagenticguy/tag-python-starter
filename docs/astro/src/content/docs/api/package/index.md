### package.hello

#### package.hello.hello_world

```python
hello_world(name='World')
```

Main entry point for the package.

**Parameters:**

- **name** (`str`) – The name to say hello to.

**Returns:**

- **str** (`str`) – The string "Hello, {name}!"

**Example:**

```python
>>> hello_world()
"Hello, World!"
```
